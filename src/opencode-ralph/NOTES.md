## What's Included

This template provides a complete development environment with:

- **Universal Linux** base image - Multipurpose Linux distribution for broad compatibility ([docs](https://github.com/devcontainers/images/tree/main/src/universal))
- **OpenCode** - AI coding assistant with chat, indexing, and memory features ([docs](https://opencode.ai/docs/))
- **Ralph** - Agentic Loop-based development ([docs](https://github.com/iyaki/ralph))
